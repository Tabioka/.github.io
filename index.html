# Enginuity Solutions Website Welcome to the Enginuity Solutions website! This is a modern, responsive, and single-page website designed to showcase the services and values of Enginuity Solutions. ## Features - **Responsive Design**: Looks great on devices of all sizes. - **Hero Section**: Captures attention with a bold welcome message and call-to-action. - **About Us Section**: Highlights the mission and expertise of Enginuity Solutions. - **Services Section**: Provides an overview of the key services offered. - **Contact Section**: Includes an email link for inquiries. ## Usage 1. Clone the repository: ```bash git clone https://github.com/yourusername/enginuity-solutions.git ``` 2. Open the `index.html` file in your browser to view the website locally. ## File Structure - `index.html`: Contains all the HTML, CSS, and JavaScript code for the website. The design is entirely self-contained. ## Technologies Used - HTML5 - CSS3 (inline styles for simplicity) - Google Fonts (Poppins) ## Deployment To deploy this website: 1. Push the code to a GitHub repository. 2. Use GitHub Pages to serve the `index.html` file: - Go to the repository's settings. - Under "Pages," select the branch containing the `index.html` file. - Save and wait for GitHub Pages to deploy your site. ## Contributions Feel free to submit pull requests to enhance the website or add new features.
